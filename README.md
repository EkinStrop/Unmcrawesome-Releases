# Unmcrawesome

MCRAW Video Editor for Android.

## Download

Check the [Releases](https://github.com/EkinStrop/Unmcrawesome-Releases/releases) page for the latest APK.

## Features

- GPU-accelerated Bayer demosaicing
- Real-time RAW video playback with color grading
- Export to H.264, HEVC (8/10-bit), ProRes, APV
- Hardware APV encoding on Snapdragon 8 Elite 5 (Currently Xiaomi 17 Ultra)
- 40+ color grading parameters
- Burst stacking and DNG extraction
